# Project Overview

Game engine for making basic board games such as Chess, Checkers

# Features to implement

 - Ability to add textures
 - Scripting
 - Grid generation and grid "snapping"
 - save/load
 - fixed camera
 - movable objects
 - add/remove pieces in-game

# Stretch goals

 - Card games
 - online/LAN multiplayer

# Libraries

 - PySide6
 - PyGame (audio)
 - Lupa (scripting)
